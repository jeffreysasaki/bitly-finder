Bit.ly finder
-
Find random bit.ly links

## Installation
- `python3 -m venv venv`
- `. venv/bin/activate`
- `pip3 install -r requirements.txt`
- `flask run`

## Usage
- http://localhost:5000 - Return 5 random bit.ly url
- http://localhost:5000?count=10 -  Return 10 bit.ly url's (May get IP banned if the count is too high)
